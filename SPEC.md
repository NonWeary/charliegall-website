# Charlie Gall Personal Website Spec

Goal:
Position Charlie as a Marketing + Data + Performance hybrid operator.

Style:
- Dark charcoal background
- Subtle green accent color
- Minimal, high-contrast
- Modern sans-serif typography
- Clean, no clutter

Sections:
1. Hero (Name, tagline, CTAs)
2. About (Marketing + analytics positioning)
3. Projects (Card grid with 5 projects)
4. Experience (Timeline)
5. Philosophy (Short performance-driven manifesto)
6. Contact (Email + LinkedIn)

Tech:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Components in src/components
- Main composition in src/app/page.tsx
